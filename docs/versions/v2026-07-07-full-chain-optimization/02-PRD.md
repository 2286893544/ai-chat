# PRD：AI Chat 全链路优化

## 功能范围

### 1. 本地数据层

- 在前端项目内新增本地数据目录。
- 目录内维护默认用户偏好、存储键、系统能力说明和角色预设。
- 运行时继续使用 IndexedDB/localStorage，不依赖远程数据库。

### 2. 后端安全与稳定性

- 支持 CORS 白名单配置。
- API 请求增加简单内存频控。
- Chat、TTS、STT 输入限制更细。
- DeepSeek、ElevenLabs、STT 服务请求增加超时。
- 核心 API 请求使用结构化校验。

### 3. 聊天上下文优化

- 上下文按消息条数和字符预算同时裁剪。
- 系统提示词保留，历史消息从最近到最旧补齐预算。

### 4. 用户体验优化

- 设置页增加本地数据、安全边界、语音能力状态说明。
- 角色管理支持导入聊天陪伴类角色预设。
- 应用新增项目说明页面或等价入口，作为轻量官网信息入口。

### 5. 文档同步

- README 说明最新能力、运行方式和本地数据边界。
- DEPLOY 说明 Render 部署、STT 限制、CORS 配置和环境变量。

## 验收标准

- `pnpm build` 可通过。
- 后端新增的可测试逻辑有自动化测试覆盖。
- 新增角色预设可以被导入或用于创建角色。
- 设置页能清楚说明本地数据和部署边界。
- 文档中不存在“远程数据库已接入”之类不符合实际的表述。

## 不做范围

- 不做服务端用户数据库。
- 不做登录注册。
- 不做云端消息同步。
- 不做支付、订阅、用量计费。
