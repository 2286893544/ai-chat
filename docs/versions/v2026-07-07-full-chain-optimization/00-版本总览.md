# AI Chat 全链路优化版本总览

文档版本：2026-07-07

版本主题：在保持轻量本地化部署的前提下，完成安全、性能、可扩展性、用户体验、部署文档和本地数据层的第一轮全链路增强。

## 背景

当前项目是个人可部署的 AI Chat 应用，采用 Vue 前端、Express API、本地 IndexedDB 数据、BYOK API Key 和可选本地 STT 服务。项目暂不引入远程数据库、账号体系、云端会话同步或计费能力。

## 本轮目标

- 保持单机和轻量云部署可运行。
- 补齐本地数据层目录，用于沉淀用户习惯、默认偏好、角色预设和本地存储键。
- 增强后端安全边界，包括 CORS、输入大小、请求频控、超时和输入校验。
- 改进聊天上下文裁剪，避免历史消息过长拖垮模型请求。
- 增强设置页和项目信息展示，让用户明确数据、语音、部署和 API Key 边界。
- 同步 README 与部署文档，保证线上说明和项目能力一致。

## 不做范围

- 不新增远程数据库。
- 不新增账号、登录、云同步、多用户权限和计费系统。
- 不把 DeepSeek 或 ElevenLabs Key 存到服务端数据库。
- 不把本地 faster-whisper STT 默认部署到 Render 免费服务。
