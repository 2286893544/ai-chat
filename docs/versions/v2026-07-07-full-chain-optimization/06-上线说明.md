# 上线说明

## 部署形态

本轮仍维持单 Render Web Service 部署：

- Express 提供 `/api/*`。
- Express 在生产环境托管 `apps/web/dist`。
- 本地 faster-whisper STT 服务不包含在 Render 免费部署内。

## 环境变量

- `DEFAULT_DEEPSEEK_MODEL`
- `DEEPSEEK_BASE_URL`
- `CORS_ORIGIN`
- `JSON_BODY_LIMIT`
- `RATE_LIMIT_WINDOW_MS`
- `RATE_LIMIT_MAX`
- `CHAT_MAX_CONTEXT_CHARS`
- `CHAT_MAX_MESSAGES`
- `CHAT_REQUEST_TIMEOUT_MS`
- `TTS_TEXT_MAX_CHARS`
- `STT_AUDIO_MAX_BYTES`
- `STT_SERVICE_URL`
- `STT_REQUEST_TIMEOUT_MS`

## 回滚

如果上线后出现 API 请求异常，可回滚到上一个 Render 部署版本。由于用户数据保存在浏览器本地，服务端回滚不迁移用户数据。
