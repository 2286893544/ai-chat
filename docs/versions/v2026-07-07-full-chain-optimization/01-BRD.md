# BRD：AI Chat 轻量本地化全链路优化

## 业务目标

让当前 AI Chat 项目从可运行 MVP 提升为可长期本地使用、可轻量部署、边界清晰的个人 AI 聊天工具。

## 用户价值

- 用户可以保留本地数据控制权。
- 用户知道 API Key、聊天记录、角色配置和语音能力的保存位置与风险。
- 用户可以从日常唠嗑、温柔陪伴、吐槽玩梗等聊天角色快速开始对话。
- 部署者可以按文档配置安全边界和线上限制。

## 成功标准

- 本地数据层目录建立，并能承载用户习惯、角色预设和存储键定义。
- 后端对 CORS、请求大小、频控、输入校验和外部请求超时有明确实现。
- 聊天上下文有预算裁剪，不再只按消息条数粗略截断。
- 设置页或应用内说明覆盖本地数据、安全边界和系统能力边界。
- README 和 DEPLOY 文档与实际能力一致。

## 业务边界

本轮只做轻量本地化增强。所有持久化数据仍优先存在用户浏览器本地，不建设远程数据库和账号体系。
